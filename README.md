# AquaCrop 敏感性分析项目

## 1. 项目概述
### 1.1 研究背景
本项目基于 AquaCrop 作物模型,实现了参数敏感性分析功能,用于评估水分限制条件下小麦生长和产量的影响因素。主要采用 EFAST (扩展傅里叶幅度敏感性测试)和 Morris 方法进行全局敏感性分析。

### 1.2 研究目标
- 识别模型中最具影响力的参数
- 评估不同气候条件下参数的敏感性变化
- 分析模型的复杂度和可塑性
- 优化区域尺度应用的参数设置

### 1.3 主要特点
- 基于 MATLAB 实现
- 支持多参数并行分析
- 提供完整的分析流程
- 可视化结果展示
- 模块化设计,易于扩展

## 2. 敏感性分析方法
### 2.1 Morris 方法
#### 原理说明
- 基于基本效应筛选
- 用于定性识别非重要参数
- 计算参数的基本效应
- 评估参数交互作用

#### 实现细节
- 轨迹数(r): 20
- 因子数(k): 等于参数数
- 阈值设置: 0.1 t/ha
- 计算 μ* 和 σ 指标

### 2.2 EFAST 方法
#### 原理说明
- 基于方差分解
- 计算主效应和总效应指数
- 定量评估参数重要性
- 分析参数交互作用

#### 实现细节
- 主效应指数(Si)计算
- 总效应指数(STi)计算
- 阈值设置: STi > 0.1
- 参数重要性排序

### 10.3 气候条件影响
#### 干旱条件特征
- 土壤水分参数(fc, pwp)敏感性增强
- 根系参数(rtx)影响加大
- 物候参数(dos, mat)保持高敏感性
- 水分胁迫系数(psen, psto)更显重要

#### 湿润条件特征
- 物候参数(dos, mat, sen, flo)更为关键
- 生长发育参数影响增强
- 土壤水分参数敏感性降低
- 温度胁迫参数作用增加

### 10.4 参数交互分析
#### 强交互参数组
1. 物候参数组
   - dos-mat: 播种日期与生长周期
   - mat-sen: 生长周期与衰老时间
   - flo-sen: 开花与衰老时间

2. 水分参数组
   - fc-pwp: 田间持水量与萎蔫点
   - fc-rtx: 田间持水量与根深
   - psen-psto: 衰老与气孔胁迫系数

#### 弱交互参数组
1. 生长参数组
   - CGC-CCx: 冠层生长与最大覆盖度
   - hi-hilen: 收获指数相关参数
   - wp-biomass: 水分生产力相关

2. 胁迫参数组
   - polmn-polmx: 授粉温度范围
   - anaer-aer: 通气相关参数

### 10.5 敏感性分类标准
#### 高敏感性
- EFAST: STi > 0.3 或 Si > 0.25
- Morris: μ* > 0.5
- 在多数气候条件下保持高敏感性
- 具有显著交互作用

#### 中等敏感性
- EFAST: 0.15 < STi < 0.3 或 0.1 < Si < 0.25
- Morris: 0.1 < μ* < 0.5
- 敏感性随气候条件变化
- 交互作用中等或较弱

#### 低敏感性
- EFAST: STi < 0.15 或 Si < 0.1
- Morris: μ* < 0.1
- 在各种条件下敏感性都低
- 交互作用不显著
