# AquaCrop 敏感性分析项目

## 1. 项目概述
### 1.1 研究背景
本项目基于 AquaCrop 作物模型,实现了参数敏感性分析功能,用于评估水分限制条件下小麦生长和产量的影响因素。主要采用 EFAST (扩展傅里叶幅度敏感性测试)和 Morris 方法进行全局敏感性分析。

### 1.2 研究目标
- 识别模型中最具影响力的参数
- 评估不同气候条件下参数的敏感性变化
- 分析模型的复杂度和可塑性
- 优化区域尺度应用的参数设置

### 1.3 主要特点
- 基于 MATLAB 实现
- 支持多参数并行分析
- 提供完整的分析流程
- 可视化结果展示
- 模块化设计,易于扩展

## 2. 敏感性分析方法
### 2.1 Morris 方法
#### 原理说明
- 基于基本效应筛选
- 用于定性识别非重要参数
- 计算参数的基本效应
- 评估参数交互作用

#### 实现细节
- 轨迹数(r): 20
- 因子数(k): 等于参数数
- 阈值设置: 0.1 t/ha
- 计算 μ* 和 σ 指标

### 2.2 EFAST 方法
#### 原理说明
- 基于方差分解
- 计算主效应和总效应指数
- 定量评估参数重要性
- 分析参数交互作用

#### 实现细节
- 主效应指数(Si)计算
- 总效应指数(STi)计算
- 阈值设置: STi > 0.1
- 参数重要性排序

## 3. 参数分析
### 3.1 参数分类
#### 保守参数
- 作物物种特定参数
- 不随品种、时间、管理变化
- 允许±33%变化范围

#### 非保守参数
- 品种特定参数
- 管理相关参数
- 环境相关参数

### 3.2 关键参数识别
#### 物候参数
- 播种日期(dos)
- 成熟所需积温(mat)
- 衰老开始积温(sen)
- 开花所需积温(flo)

#### 水分相关参数
- 田间持水量(fc)
- 萎蔫点(pwp)
- 最大有效根深(rtx)
- 冠层衰老水分胁迫系数(psen)
